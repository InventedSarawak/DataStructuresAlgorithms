## 🚀 Quick Start

If you’d like to run a solution locally:
1. **Clone the repo**:  
   ```bash
   git clone https://github.com/InventedSarawak/Data-Structures-Algorithms.git
   cd leetcode-dsa-practice
   ```
2. **Run any file** with Python:
   ```bash
   python3 two_sum.py
   ```
